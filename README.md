# Talon - Daily & Achievement Monitor
**Version:** 2.4.0

## Description
Talon is a lightweight World of Warcraft addon designed to help players track specific daily quests and achievements across various expansions. It intelligently scans all relevant maps for available quests related to tracked achievements and filters out those you have already completed. It also allows for custom quest tracking by Quest ID or Achievement ID.

## What's New in v2.4.0
*   **UI Overhaul**: Fixed layout issues where expansion filters and scan results were hidden.
*   **Achievement ID Tracking**: You can now input an Achievement ID in the settings to automatically find and track its related uncompleted quests.
*   **Manual Scan**: Added a "Manual Scan" button directly to the configuration panel.
*   **Localization**: Improved translations and UI text for multiple languages.

## Features
*   **Smart Scanning**: Automatically detects available daily quests for tracked achievements in all zones after 7.0.
*   **Advanced Custom Tracking**: Add specific quest IDs, or simply enter an **Achievement ID** to automatically find and track relevant uncompleted quests.
*   **Progress Tracking**: Filters out completed quests and achievements so you only see what needs to be done.
*   **Sound Alerts**: Optional audio notification when a relevant quest is found.
*   **Modern Interface**: Fully integrated with the WoW 11.0+ Settings panel, featuring a manual scan button and session results list.
*   **Multi-Expansion Support**: Includes data for The War Within, Dragonflight, Shadowlands, Battle for Azeroth, Legion, Warlords of Draenor, and Cataclysm.

## Slash Commands
*   `/talon config` - Open the configuration panel.
*   `/talon scan` - Manually scan for available quests across all tracked maps.

## Installation
1.  Download the addon.
2.  Extract the `Talon` folder to your World of Warcraft `Interface\AddOns` directory.
3.  Reload your UI or restart the game.

# Talon (鹰爪) - 成就与日常监控助手
**版本:** 2.4.0

## 插件简介
Talon 是一款轻量级的魔兽世界插件，旨在帮助玩家高效追踪跨资料片的特定日常任务和成就。它能够智能扫描所有相关地图，查找与追踪成就相关的可用任务，并自动过滤掉已完成的内容。同时支持通过任务ID或成就ID进行自定义监控。

## v2.4.0 更新内容
*   **界面修复**：修复了资料片过滤器和扫描结果列表可能被隐藏的布局问题。
*   **成就ID追踪**：现在可以直接在设置中输入成就ID，插件会自动查找并追踪其关联的未完成任务。
*   **手动扫描**：选项面板新增了“手动扫描”按钮，方便随时检查。
*   **本地化**：优化了多语言翻译和界面文本显示。

## 功能特性
*   **智能扫描**：自动检测7.0以后各个版本与追踪成就相关的可用日常任务。
*   **高级自定义追踪**：在设置面板中添加特定任务ID，或直接输入**成就ID**自动查找并追踪相关的未完成任务。
*   **进度追踪**：过滤已完成的任务和成就，只显示未完成的目标。
*   **声音提示**：发现相关任务时播放音效提醒（可选）。
*   **现代化界面**：完全集成至魔兽世界 11.0+ 设置面板，包含手动扫描按钮和本次扫描结果列表。
*   **多资料片支持**：涵盖地心之战 (11.0)、巨龙时代、暗影国度、争霸艾泽拉斯、军团再临、德拉诺之王及大灾变等内容。

## 命令列表
*   `/talon config` - 打开配置面板。
*   `/talon scan` - 手动扫描所有追踪地图的可用任务。

## 安装说明
1.  下载插件包。
2.  将 `Talon` 文件夹解压至您的魔兽世界 `Interface\AddOns` 目录下。
3.  重载界面 (`/reload`) 或重启游戏即可。
